# GoogleDriveSender

アップロード対象フォルダをZIP化→アップロード→ドメイン共有→共有リンクをクリップボードコピーします  
※動作にはAPIの認証ファイル（credentials.json）を実行フォルダと同階層に配置する必要があります  
参考（https://developers.google.com/drive/api/v3/quickstart/dotnet）
